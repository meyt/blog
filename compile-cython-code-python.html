کامپایل کد های پایتون

Sun 11 January 2015 توسط MeyTGHG

با استفاده از برنامه cython کد های پایتون رو میشه به c تغییر داد و کامپایل کرد و در نهایت در پروژه import ! خیلی دنبال همچین چیزی بودم و تشکر میکنم از دوست خوبم اقا وحید که این رو معرفی کردن.

$ sudo pip install cython

فایلی که قصد کامپایل کردن‌اش رو دارید به این صورت وارد کنید تا تبدیل به کد c بشه ::

$ cython -a mytest.pyx

محتوای فایل mytest.pyx 

def fib(n):
    """Print the Fibonacci series up to n."""
    a, b = 0, 1
    while b < n:
    print b,
    a, b = b, a + b

واما کامپایل :) 

$ gcc -shared -pthread -fPIC -fwrapv -O2 -Wall -fno-strict-aliasing -I/usr/include/python2.7 -o mytest.so mytest.c
